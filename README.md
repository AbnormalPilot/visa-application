# Visa Application Portal

This project is a web-based portal for visa applications, created as a first-year major project. It is built using only HTML, CSS, and JavaScript.

## Status

**Under Development**

## Features

*   User-friendly interface for filling out visa application forms.
*   Client-side form validation to ensure data accuracy.
*   Responsive design for accessibility on various devices.
*   A multi-step application process to guide users.

## Technologies

*   **HTML5:** For the structure and content of the web pages.
*   **CSS3:** For styling and layout.
*   **JavaScript:** For interactivity and client-side logic.

## Development Plan

1.  **Phase 1: Basic Structure (HTML)**
    *   Create the main `index.html` file with the basic layout.
    *   Develop the HTML structure for the application form, header, and footer.
    *   Set up different sections of the form (e.g., personal details, travel information).

2.  **Phase 2: Styling (CSS)**
    *   Create `css/style.css` to define the visual appearance.
    *   Style the form elements for better user experience.
    *   Implement a responsive design using media queries.

3.  **Phase 3: Interactivity (JavaScript)**
    *   Create `js/script.js` for client-side scripting.
    *   Implement form validation to check for required fields and correct data formats.
    *   Develop the logic for the multi-step form.
